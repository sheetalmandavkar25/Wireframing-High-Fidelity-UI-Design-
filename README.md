# Food Court – High-Fidelity UI Design

## Project Overview

Food Court is a high-fidelity mobile application UI design created using Figma as part of the CodeAlpha UI/UX Design Internship. The project transforms low-fidelity wireframes into visually appealing and user-friendly interfaces by incorporating color palettes, typography, icons, buttons, and images.

The application is designed to provide users with a seamless food ordering experience, from browsing menu items to completing payment and receiving order confirmation.

---

## Objective

To design a modern and intuitive food delivery application interface while applying fundamental UI/UX principles such as consistency, visual hierarchy, usability, and user-centered design.

---

## Tool Used

* Figma

---

## Design Features

### Color Palette

* Primary Color: Orange
* Secondary Colors: White and Gray
* Consistent branding across all screens

### Typography

* Clear and readable text hierarchy
* Bold headings for important information
* Simple and user-friendly font styles

### Visual Elements

* Food images
* Navigation icons
* Rounded buttons and input fields
* Consistent spacing and alignment

## Screens Included

### 1. Splash Screen

* Application logo
* Welcome message
* Get Started button

### 2. Login Screen

* Email and password input fields
* Forgot Password option
* Login button
* Sign Up option

### 3. Home Screen

* Personalized greeting
* Search bar
* Food categories
* Recommended food items
* Bottom navigation menu

### 4. Food Details Screen

* Food image
* Price and rating
* Product description
* Add to Cart button

### 5. Cart Screen

* Selected item details
* Quantity and total amount
* Payment mode selection
* Pay Now button

### 6. Order Confirmation Screen

* Order confirmation message
* Order ID
* Estimated delivery time
* Track Order option
* Back to Home button

## User Flow

Splash Screen → Login → Home → Food Details → Cart → Order Confirmation
